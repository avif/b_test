Info

Heroku uploaded version https://budgeto.herokuapp.com

Local usage
------------

Clone the repo.

Install npm dependencies

    npm install

Install bower dependencies

    bower install

Run mongo

    mongod [path/to/mongod.exe]

Run Grunt 'serve' task

    grunt serve
